For the coup de grâce, let's wrap the whole thing in a function.

Write a function gradient_descent, which takes three arguments:

  1. the initial value x,
  2. the learning rate eta
  3. and the maximum number of iterations max_iter.

It should then update the value x according to Gradient descent (see loop above) and finally return the approximation to the minimum.

In [16]:
Student's answer(Top)
def gradient_descent(x, eta, max_iter):
    
    for _ in range(max_iter):
        gradient = dfdx(x)
        if abs(gradient) < 1e-5: break
        x -= eta * gradient
        
    return x

# function makes a plot that shows where different starting points converge

# need this numpy setting so I can catch the numpy RuntimeWarning when dfdx diverges
# doing it like this so I do not need to rewrite the dfdx function to stop divergences
np.seterr(all='raise')
def plotConvergence(xrange, eta, max_iter, tolerance):
    
    # define start points and bins into which to categorise them
    startingPoints = np.linspace(-xrange, xrange, 400)
    convergeLeft = []
    convergeRight = []
    convergeSlow = []
    diverge = []

    for sp in startingPoints:
        
        # bit nasty but using this to detect divergences
        try:
            gd = gradient_descent(sp, eta, max_iter)
        except:
            diverge.append(sp) 
            continue
        
        # sort into remaining lists
        if -1 - tolerance < gd < -1 + tolerance:
            convergeLeft.append(sp)
        elif 1 - tolerance < gd < 1 + tolerance:
            convergeRight.append(sp)
        else:
            convergeSlow.append(sp)
    
    # plot te diagram
    plt.plot(startingPoints, f(startingPoints), c='k', zorder=100)
    plt.scatter(convergeLeft, f(np.array(convergeLeft)),c='C3', marker='x', label="converge at -1")
    plt.scatter(convergeRight, f(np.array(convergeRight)),c='C0', marker='x', label="converge at +1")
    plt.scatter(convergeSlow, f(np.array(convergeSlow)),c='C4', marker='x', label="max_iter reached before tolerance achieved")
    plt.scatter(diverge, f(np.array(diverge)),c='C2', marker='x', label="diverge to infinity")
    plt.legend()
    plt.title(f'$\eta = {eta:.4f}$, max_iter = {max_iter}, tol = {tolerance:.1e}')
    plt.show()

$\eta$ too large¶

From the following plot we can see that for starting points where the slope is very steep, the first iteration step skips over the minima and either hits the minimum on the other side, or overshoots so far that the iteration diverges off to infinity. Making $\eta$ larger brings the point at which divergence happens closer to 0 (compare first and second plots). From the second plot we can see that increasing the number of iterations does not help in this case, as this only makes the diverging points go even further from the minima

Finding $\eta$ such that we only need 1 step¶

Essentially need to solve the following equation for $\eta$ $$ \left|x_0 - \eta \left.\frac{df}{dx}\right|_{x_0}\right| = 1 $$

which gives

$$ \eta = \frac{|x_0| - 1}{4|x_0|(x_0^2 - 1)} $$

Which means that for any starting point $x_0$ (except $x_0 = 0$) we can find an $\eta$ that would converge the iteration in 1 step. In the graph below we see that for $x_0 = \pm 1.5$ we can reach the minima in 1 step using $\eta = \frac{2}{30}$
